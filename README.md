# TestRademade
Test task for Rademade
## Getting Started
These instructions will help you to copy the project and run it on your local machine
### Installing
1. Install node.js and npm
* [npm](https://www.npmjs.com/get-npm) - download and install npm.
2. Go to project root and install node_modules
```
npm i
```
### Starting gulp tasks
There are several gulp tasks that you may need:
1. Run gulp mainTask to let gulp watch your files and compile them to dist, when you make changes
```
gulp mainTask
```
2. Run gulp clean:dist to make dist folder empty
```
gulp clean:dist
```
2. gulp compressImages - for compressing all images that are in src/img folder
```
gulp compressImages
```

**Hope everything will work fine**
